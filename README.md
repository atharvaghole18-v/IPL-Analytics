# 🏏 IPL Analytics Dashboard

An end-to-end data analytics project analyzing 10+ years of IPL cricket data.

## 📊 Features
- Team win analysis across all seasons
- Top run scorers and wicket takers
- Toss decision impact analysis
- Season-wise match trends
- Interactive filters by season
- Live KPI cards

## 🛠️ Tech Stack
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Streamlit

## 📁 Dataset
IPL Complete Dataset (2008–2023) from Kaggle

## ▶️ How to Run
pip install -r requirements.txt
streamlit run dashboard/app.py
